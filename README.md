# Lingo-Bridge
🌐 AI-Powered Language Translator is a web-based app that helps you translate text between multiple languages like English, Italian, Russian, Spanish, and French. Built using HTML, CSS, and JavaScript, it features a clean, modern design with a soft lavender and burgundy color theme for an appealing user experience 🎨. You can either type your message ✍️ or use voice-to-text 🎤 thanks to built-in speech recognition. The translations are powered by public APIs, ensuring quick and accurate results ⚡. Whether you're a student, traveler, or just someone curious about languages, this tool makes breaking language barriers easy and fun. Say hello to effortless communication, one translation at a time! 🌎💬
